# Heroes & Monsters Web

这是对 `valogame-base` 中 `Heroes and Monsters` 经典模式的网页化改编版。

## 当前玩法

- 选择最多 3 名英雄开局
- 在 `8x8` 地图上移动
- 普通地块会触发随机事件
- 市场地块可以购买装备、护甲、药水和法术
- 普通地块可能触发怪物遭遇
- 战斗采用浏览器内回合制弹层
- 战斗胜利后获得金币和经验

## 和原 Java 版的关系

- 保留了职业、地图、市场、掉落、战斗这些核心元素
- 改掉了终端 / ASCII 呈现方式
- 改为浏览器卡片式 UI
- 当前是可玩 MVP，不是原游戏 1:1 完整重写

## 当前未覆盖

- 完整经济系统
- 更细的职业成长差异
- 更复杂的战斗菜单
- `Legends of Valor` 模式

## 本地状态

浏览器会把当前运行状态保存到 `localStorage`，刷新页面后会继续上一次进度。
